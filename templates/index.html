Mathematical Discussion

Euler’s Method is a numerical technique used to approximate solutions of first-order differential equations.

It uses the slope of the function to estimate the next value.

\[ y_{n+1}=y_n+h f(x_n,y_n) \]

Where:

  • yₙ = current y value
  • h = step size
  • f(xₙ,yₙ) = derivative equation

Worked Example 1

\[ \frac{dy}{dx}=x+y \]
  • x₀ = 0
  • y₀ = 1
  • h = 0.1

Step 1

f(0,1) = 1

y₁ = 1 + 0.1(1)

y₁ = 1.1

Step 2

f(0.1,1.1) = 1.2

y₂ = 1.1 + 0.1(1.2)

y₂ = 1.22

Final Answer

\[ y(0.2)\approx1.22 \]

Worked Example 2

\[ \frac{dy}{dx}=x-y \]
  • x₀ = 0
  • y₀ = 2
  • h = 0.2

Step 1

f(0,2) = -2

y₁ = 2 + 0.2(-2)

y₁ = 1.6

Step 2

f(0.2,1.6) = -1.4

y₂ = 1.6 + 0.2(-1.4)

y₂ = 1.32

Final Answer

\[ y(0.4)\approx1.32 \]
